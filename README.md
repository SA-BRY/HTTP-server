
# 🚀 TypeScript HTTP Server (From Scratch)

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.9. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.


A minimal HTTP server built from scratch using raw TCP sockets in TypeScript.
This project is for learning how HTTP works at a low level.

---

## ✨ Features

* Raw TCP socket server
* Manual HTTP request parsing
* Basic HTTP response handling
* Lightweight and dependency-free
